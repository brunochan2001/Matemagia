Area del Circulo

Para calcular el área de un círculo debes antes obtener o calcular el radio, es decir, la distancia desde el centro del círculo hacia el exterior.

La fórmula que debes aplicar para calcular el área de un círculo es:
A = Π x r²

En este artículo de como te explicamos paso a paso cómo sacar el área de un círculo.

Area Del Cuadrado

El área de un cuadrado se calcula a partir de uno de sus lados (a).

Es el producto de la base por la altura del cuadrado, ya que al ser ambas iguales, el área será un lado al cuadrado.
